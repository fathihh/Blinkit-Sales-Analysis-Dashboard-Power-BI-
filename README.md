# 📊 Blinkit Sales Analysis Dashboard (Power BI)

## 🔍 Overview
This project presents an interactive **Power BI dashboard** developed to analyze and visualize sales performance using Blinkit grocery data. The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance and trends  
- Identify top-performing product categories  
- Evaluate outlet performance across location, size, and tier  
- Deliver an interactive and user-friendly reporting solution  

---

## 📌 Key Metrics
- **Total Sales:** $1.20M+  
- **Total Items:** 8,500+  
- **Average Sales:** $141  
- **Average Rating:** 3.9  

---

## 📈 Features & Insights
- Sales trend analysis by outlet establishment year  
- Product performance by **Item Type** and **Fat Content**  
- Outlet segmentation by **Size, Location, and Tier**  
- KPI cards for quick performance tracking  
- Interactive filters for dynamic analysis  

---

## 🛠️ Tools & Technologies
- **Power BI** – Data Visualization  
- **Power Query** – Data Cleaning & Transformation  
- **DAX (Data Analysis Expressions)** – KPI & Measures  
- **Data Modeling** – Star Schema Design  

---

## 🧱 Data Model
The dataset follows a **star schema** approach:

- **Fact Table:** Sales Data  
- **Dimension Tables:**  
  - Outlet  
  - Item  
  - Location  

---


## 📊 Key Learnings
- Built optimized data models for better performance  
- Developed DAX measures for dynamic calculations  
- Designed business-focused and interactive dashboards
